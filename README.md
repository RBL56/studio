# Deriv Trading Bot Builder

This is a Next.js application for building and backtesting trading bots for the Deriv platform.

## Features

- **Visual Bot Builder**: Create trading strategies using natural language.
- **Quick Start Templates**: Get started quickly with pre-built strategies.
- **AI-Assisted Backtesting**: Simulate your strategy's performance against historical data and get AI-powered insights.
- **Tutorials & Guides**: Learn the basics of automated trading.

## Getting Started

To get started, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

The main application is located in `src/app/page.tsx`.
